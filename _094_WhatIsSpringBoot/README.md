**What is Spring?**
-
Set of tools that helps you write java faster.

What Spring Can Do?
-

**Microservices**
- Quickly deliver production-grade features with independently evolvable microservices.

**Reactive**
- Spring's asynchronous, nonblocking architecture means you can get more from your computing resources

**Cloud**
- You code, any cloud -- Spring's got you covered.Connect and scale your services, whatever your platform.

**Web Apps**
- Frameworks for fast, secure, and responsive web applications connected to any data store.

**Serverless**
- The ultimate flexibility. Scale up on demand and scale to zero when there's no demand.

**Event Driven**
- Integrate with your enterprise. React to business events. Act on your streaming data in real time.
